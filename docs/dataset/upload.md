## 上传数据

> 在启智平台上，你可以在项目下创建数据集，再将数据文件打包上传。

<!-- tabs:start -->

#### **数据集页面**

点击 **创建数据集**，为项目创建数据集。

项目与数据集是绑定的，每个项目只能创建一个数据集，但是你可以在数据集里上传多个打包文件。

<img src="_media/dataset/uploadhome.png" width = "800" alt="traindetail" align=middle />

#### **创建数据集**

创建详情界面，数据集名称只能包括 `字母，数字，_和-`。创建成功后，你仍可以修改数据集名称，说明，以及标签。

数据集与项目绑定，一旦创建便无法单独删除。但你可以通过删除打包文件与修改数据集来更新你的数据集。

<img src="_media/dataset/uploadcreate.png" width = "800" alt="traindetail" align=middle />

<img src="_media/dataset/uploadempty.png" width = "800" alt="traindetail" align=middle />

#### **文件上传**

请注意，打包文件的格式为 `.zip` `.tar` `.gz`。

选择你的代码环境 `CPU/GPU` 或 `NPU` 。然后点击方块选择本地文件，或者将文件直接拖拽到方框内。

点击 **上传** 开始上传，上传成功将自动跳转到数据集界面。
